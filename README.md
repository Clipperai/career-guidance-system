# Career Guidance System

A simple Numpy based career recommendation system that suggests the best career paths based on:

- Academic performance
- Skills
- Personal interests

## Features

- Student data input
- Skill analysis
- Interest-based career mapping
- Score-based ranking
- Top 3 career suggestions

## Tech Stack

- Python
- NumPy

## Run Project

git clone https://github.com/Clipperai/career-guidance-system

cd career-guidance-system/src

python career_guidance.py
